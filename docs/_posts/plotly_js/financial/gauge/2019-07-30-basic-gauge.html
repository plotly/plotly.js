--- name: Basic Gauge language: plotly_js suite: gauge-charts order: 1 sitemap: false arrangement: horizontal markdown_content: | A radial gauge chart has a circular arc, which displays a single value to estimate progress toward a goal. The bar shows the target value, and the shading represents the progress toward that goal. Gauge charts, known as speedometer charts as well. This chart type is usually used to illustrate key business indicators. The example below displays a basic gauge chart with default attributes.
