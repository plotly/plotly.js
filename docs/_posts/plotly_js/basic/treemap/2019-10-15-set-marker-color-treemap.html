--- name: Set Color of Treemap Sectors language: plotly_js suite: treemap order: 3 sitemap: false arrangement: horizontal markdown_content: | There are three different ways to change the color of the sectors in Treemap: 1) [marker.colors](https://plotly.com/javascript/reference/treemap/#treemap-marker-colors), 2) [colorway](https://plotly.com/javascript/reference/layout/#layout-colorway), 3) [colorscale](https://plotly.com/javascript/reference/treemap/#treemap-marker-colorscale).
