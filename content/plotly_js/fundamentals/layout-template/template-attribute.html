--- name: The Layout Template Attribute language: plotly_js suite: layout_template order: 0 sitemap: false arrangement: horizontal markdown_content: | The `template` attribute of `layout` allows a Plotly chart to take it's style and formatting from a `template` object. `template`s can be generated using [Plotly.makeTemplate](https://plotly.com/javascript/plotlyjs-function-reference/#plotlymaketemplate) or manually. `annotaions`, `updatemenus`, `images`, `shapes` and other container array objects in the Plotly `layout` are specially handled by the template machinery to provide more flexibility when using these container arrays in plots derived from these templates. For more information see [https://plotly.com/javascript/reference/layout/#layout-template](https://plotly.com/javascript/reference/layout/#layout-template). ---
